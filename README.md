# I — Your Personal Health OS

A premium futuristic multilingual MVP built with Next.js, TypeScript, and TailwindCSS.

## Included MVP scope

- Cinematic landing page for **I — Your Personal Health OS**
- RTL-first default experience (Hebrew)
- Dynamic language switching: Hebrew, English, Spanish, French, German, Arabic, Russian
- LTR/RTL direction switching based on selected locale
- Dark/light theme toggle (dark as default)
- Dashboard command-center mock with modular health cards

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
